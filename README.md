PROJECT DESCRIPTION
This is a three page event website which was created as part of the INS 322 Web Design and Management CAT 2 project assignment.
The website presents the different the different kind of events available for students in a gallery format, and an event schedule is provided in the form of a table. Moreover, the website contains two kinds of forms, one being the subscribe form and the sign in form.
It has also been structured using HTML and styled using CSS to make it presentable although its functionality is very limited due to the lack of Javascript in it.

PAGE DESCRIPTIONS
1.index.html(Home page)
  - This page works as the home page for the website.
  - It contains a navigation bar that links all the three pages, which is also fixed and with a Z-index of 1 to enable navigation from anywhere in the website's home page. 
  -An image on the main section of the main page has been used as the background decoration of the website's introduction words.
  - More images have been used both for visual appearance and to also provide more information about the specific events that they are tagged with. The images also work as links to the booking information of the tickets etc, which are external websites.
  A table presents the individual schedule in the events that they are interested in.
  
2.form.html(Sign up page)
  - This is the sign up page with a link to the sign in page for the users who have been in the website before.
  - The page requires the user to input their username, email addresses and passwords as their required fields.
  
3.contact.html(Contact page)
  - This is the contact page that contains all the information on whom to contact depending on the type of issue.
  - it also has a form to fill in as a faster resolution to using the contact table.
  
HOW TO RUN THE WEBSITE
1. Ensure that all the files are in their correct order as shown in the structure above.
2. Open index.html in any modern web browsers such as chrome, firefox, safari etc.
3. The navigation page at the top of the index.html(home page) and the links within the other websites can be used to interact with the different web pages.
4.No internet connection is required to access the website as all files are local.
